# SillyTavern 工具箱

一个实用的 SillyTavern 输入框工具类插件，在输入框上方提供快捷工具按钮。

## 功能

- **时间戳** 🕐 - 一键插入当前时间 `[14:30:45]`
- **日期时间** 📅 - 插入完整日期时间 `2026/5/16 14:30:45`
- **复制上一条** 📋 - 将AI上一条消息复制到输入框
- **清空** 🗑 - 清空输入框内容
- **转为大写** ABC - 将文本转为大写
- **转为小写** abc - 将文本转为小写
- **去除空格** ✂ - 去除首尾空格

## 安装

### 方式一：使用扩展安装器

1. 打开 SillyTavern
2. 进入设置页面
3. 点击"扩展"选项卡
4. 使用扩展安装器安装本插件

### 方式二：手动安装

1. 将插件文件夹复制到 `data/<your-user>/extensions/third-party/st-toolbox`
2. 重启 SillyTavern

## 使用

1. 安装完成后，在设置页面的"扩展"选项卡中找到"工具箱"面板
2. 勾选"启用工具箱"以显示工具栏
3. 可以根据需要勾选/取消勾选各个工具按钮

## 截图

工具栏将固定显示在输入框上方：
```
[🕐] [📅] [📋] [🗑] [ABC] [abc] [✂]
                              ↓
                    [输入框]
```

## 设置说明

| 设置项 | 说明 | 默认 |
|--------|------|------|
| 启用工具箱 | 显示/隐藏整个工具栏 | ✓ |
| 时间戳按钮 | 显示/隐藏时间戳按钮 | ✓ |
| 日期时间按钮 | 显示/隐藏日期时间按钮 | ✓ |
| 复制上一条按钮 | 显示/隐藏复制按钮 | ✓ |
| 清空输入框按钮 | 显示/隐藏清空按钮 | ✓ |
| 转为大写按钮 | 显示/隐藏大写按钮 | ✗ |
| 转为小写按钮 | 显示/隐藏小写按钮 | ✗ |
| 去除首尾空格按钮 | 显示/隐藏修剪按钮 | ✓ |

## 技术细节

- 使用原生 JavaScript 和 jQuery
- 兼容移动设备
- 支持深色/浅色主题
