# 智慧触控屏

一个智能的 SillyTavern 工具栏插件，在输入框上方提供强大的辅助功能。

## 功能特性

### 📝 生成回复
- 一键生成多条不同风格的回复选项
- 支持选择生成数量（1-3条）
- 点击即可发送选中的回复

### 📚 世界书管理
- 自动分析对话提取出场人物
- 生成包含角色简介的世界书条目
- 一键保存到世界书

### 📊 对话总结
- 智能总结最近的对话内容
- 支持一键发送或复制

### 🔍 角色分析
- 基于角色设定和对话历史分析性格
- 提供深入的角色洞察

### 💡 情节建议
- 根据当前对话提供情节发展建议
- 帮助推动故事进展

## 安装方法

### 方式一：使用扩展安装器

1. 打开 SillyTavern
2. 进入设置页面
3. 点击"扩展"选项卡
4. 使用扩展安装器安装本插件

### 方式二：手动安装

1. 将插件文件夹复制到 `data/<your-user>/extensions/third-party/st-smart-toolbar`
2. 重启 SillyTavern

## 使用说明

1. 安装完成后，在设置页面的"扩展"选项卡中找到"智慧触控屏"面板
2. 勾选"启用智慧触控屏"以显示工具栏
3. 点击工具栏上的功能按钮进入对应功能页面
4. 在功能页面中完成操作后，点击返回按钮或发送后自动返回

## 设置说明

| 设置项 | 说明 | 默认值 |
|--------|------|--------|
| 启用智慧触控屏 | 显示/隐藏整个工具栏 | ✓ |
| 发送后自动返回 | 发送消息后自动返回主界面 | ✓ |
| 默认生成回复数量 | 设置默认生成的回复条数 | 3条 |

## 技术细节

- 基于 SillyTavern 官方扩展模板开发
- 使用原生 JavaScript 和 jQuery
- 兼容最新版 SillyTavern
- 纯CSS动画效果，性能优秀
